# 스포츠 용품 역경매 기반 거래 플랫폼

## 1. 프로젝트 소개
스포츠 용품을 대상으로 한 **역경매(Reverse Auction) 기반 웹 거래 플랫폼**입니다.  
구매자가 원하는 스포츠 용품을 요청하면, 여러 판매자가 해당 요청에 대해 **입찰(제안)**을 등록하고,  
구매자가 그 중 하나를 선택해 거래를 진행하는 구조입니다.  

MVP 시연 플로우:
- 회원가입 / 로그인
- 역경매 상품(구매 요청) 목록 조회
- 특정 요청에 대한 입찰 등록
- 마이페이지에서 나의 경매 / 입찰 내역 확인
- 커뮤니티 게시판에서 글 작성 / 조회
- 마이페이지에서 나의 글 목록 확인
- 내 정보 및 배송지 수정

## 5. 주요 기능

### 5-1. 회원 / 인증
- 회원가입: 기본 회원 정보 등록, 유효성 검증
- 로그인 / 로그아웃: 세션 또는 토큰 기반 인증
- 마이페이지 진입을 위한 인증 처리

### 5-2. 역경매(Reverse Auction) 기능
- 역경매 요청 생성 (구매자)
  - 스포츠 용품 카테고리 선택
  - 요청 제목/내용, 희망 최대가, 희망 조건, 입찰 마감 시간, 결정 기한 설정
- 역경매 요청 목록 조회
  - 카테고리 / 키워드 / 정렬(마감 임박, 입찰 수 등) 필터링
- 역경매 요청 상세 조회
  - 요청 상세 정보, 입찰 현황, 남은 시간, 상태 표시(진행중/마감/유찰 등)

### 5-3. 입찰(Bid) 기능 (판매자)
- 입찰 등록
  - 제안 가격, 수량, 메시지 입력
- 입찰 내역 조회
  - 특정 역경매에 대한 입찰 리스트 (가격, 입찰자, 시간, 상태)
- 낙찰 처리(비즈니스 로직)
  - 구매자가 특정 입찰을 선택해 낙찰 처리하는 흐름 설계

### 5-4. 마이페이지
- 나의 경매 내역 보기 (구매자로서)
  - 내가 생성한 역경매 요청 목록
  - 상태, 입찰 수, 마감일, 상세보기
- 나의 입찰 내역 보기 (판매자로서)
  - 내가 입찰한 역경매 목록
  - 나의 입찰 가격, 결과(낙찰/실패) 표시
- 내 정보 관리
  - 닉네임, 이메일, 전화번호, 비밀번호 변경 등
- 배송지 관리
  - 배송지 등록/수정/삭제
  - 대표 배송지 설정

### 5-5. 커뮤니티 / 게시판
- 게시판 목록 조회
  - 게시판(골프/스키 등) 별 글 목록
  - 제목, 작성자, 조회수, 댓글 수, 작성일 표시
- 글쓰기
  - 게시판 선택, 제목, 내용, 이미지 첨부
- 글 상세보기
  - 본문, 이미지, 조회수, 댓글 리스트
- 나의 글 내역 보기 (마이페이지)
  - 내가 작성한 게시글 목록 조회
